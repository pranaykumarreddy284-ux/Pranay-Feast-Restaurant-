#  Restaurant Website (HTML & CSS)

This project is a **Restaurant-style website** built using **pure HTML and CSS**.  
It is designed to look clean, modern, and professional, similar to real restaurant websites.

---

##  Features

- Clean and modern UI design  
- Built using only **HTML5** and **CSS3**  
- Responsive layout (Desktop & Mobile friendly)  
- Single HTML file structure  
- Hero section with background image  
- About section  
- 21 Menu items with images  
- All food images in same size  
- Contact section at the bottom  
- Beginner-friendly frontend project  

---

##  Technologies Used

- HTML5  
- CSS3  
- Visual Studio Code
